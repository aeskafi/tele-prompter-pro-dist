# 🎥 TelePrompter Pro

 **The professional, market-ready teleprompter solution for creators, speakers, and educators.**

 TelePrompter Pro is designed to provide a seamless script-reading experience with advanced controls, cross-device synchronization, and AI-powered tools. Built for
      performance and reliability on Windows, macOS, and Linux.

 ---

 ## 🚀 Key Features

 - **Smooth Scrolling:** High-precision, adjustable speed control for natural reading.
 - **Rich Text Support:** Import and edit scripts with full formatting (Bold, Italic, Colors).
 - **Remote Control:** Control your prompter from a smartphone or another browser window.
 - **AI Translation:** Instant script translation into multiple languages.
 - **Mirror Mode:** Flip text horizontally or vertically for use with professional prompter glass.
 - **Offline First:** Desktop application works perfectly without an internet connection.

 ---

 ## 📥 Download & Installation

 You can find the latest stable releases in the **[Releases](https://github.com/aeskafi/tele-prompter-pro-dist/releases)** section.

 ### Windows
 1. Download `TelePrompter-Pro-Setup-1.0.4.exe`.
 2. Run the installer and follow the on-screen instructions.

 ### macOS
 1. Download `TelePrompter-Pro-1.0.4-arm64.dmg` (for Apple Silicon) or the Intel version if applicable.
 2. Open the `.dmg` file and drag **TelePrompter Pro** to your Applications folder.
 3. *Note: You may need to right-click and select "Open" for the first time if the app is not notarized.*

 ### Linux
 - **Debian/Ubuntu:** Download the `.deb` package and install via `sudo dpkg -i ...`
 - **Universal:** Download the `.AppImage`, make it executable (`chmod +x`), and run it directly.
